# Elder Care Agent MVP

Monorepo including:

- `apps/api-server`: FastAPI backend, scheduler, DB models, uploads
- `apps/mcp-server`: Python MCP server exposing database-backed tools
- `packages/agent-core`: LangGraph agents and providers
- `apps/elder-web`: elderly-facing voice-only UI
- `apps/family-web`: family dashboard and upload center
- `packages/shared-types` and `packages/ui`: frontend shared packages
